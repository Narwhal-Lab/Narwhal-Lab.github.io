Directions

招募方向

01

Physical AI

面向物理规律、因果一致性和世界模型驱动的具身智能系统。

  • Vision-Language-Action Model
  • World Action Model
  • Robotics Foundation Model
  • Physics-Grounded Agent
  • 强化学习、模仿学习、逆强化学习
02

Agentic Efficiency

面向大模型与智能体系统的效率优化，以及可持续 AI 系统设计。

  • LLM / World Model / VLA 推理加速
  • 模型压缩、剪枝、量化、蒸馏与后训练
  • 面向 token、算力、延迟、能耗的推理优化
  • 工具调用、记忆、检索与缓存机制优化
03

Agentic Safety & Security

面向智能体系统在真实应用中的安全性、鲁棒性与可信性。

  • AI 代码生成安全
  • Agent 应用安全与工具调用安全
  • 具身智能安全与风险控制
  • 多智能体系统攻击、防御与协同安全
04

Related Topics

也欢迎对智能体自进化、后训练、多智能体协作、Agent Benchmark、开源智能体系统、AI for Science / Engineering 感兴趣的同学交流。

Profile

适合这些同学

基本要求

  • 对上述方向有浓厚兴趣，并希望通过科研训练提升未来深造或研究竞争力
  • 具备扎实的计算机基础，熟悉机器学习或深度学习基础
  • 有较强的代码实现能力、动手能力和执行力
  • 能够长期稳定投入科研工作，建议合作周期不少于 6 个月，每周可投入 15-20 小时以上者优先
  • 能够阅读英文论文，愿意参与论文复现、方法改进、实验分析和学术写作

加分项

  • 有机器学习、深度学习、多模态学习、大语言模型、世界模型、具身智能或智能体相关项目经验
  • 有算法实现、科研训练、论文阅读、学术写作或论文发表经验
  • 熟悉 Python、PyTorch、Hugging Face、LLM / VLM / VLA 框架或 Agent 开发平台
  • 有 AI-assisted coding、开源项目开发或系统工程经验
  • 有数学建模、ACM、Kaggle、机器人、自动驾驶、多智能体等相关竞赛经历
